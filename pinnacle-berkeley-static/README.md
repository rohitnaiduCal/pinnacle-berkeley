# Pinnacle @ Berkeley - Static Website

A clean, modern landing page for Pinnacle @ Berkeley - Elite AI Consulting.

## Features

- Fully responsive design
- Animated revenue counter
- Infinite scrolling logo carousel
- Smooth scroll animations
- Interactive hover effects

## Files

- `index.html` - Main HTML structure
- `styles.css` - All styling and animations
- `script.js` - Interactive functionality

## GitHub Pages Setup

This site is configured to work with GitHub Pages. Simply:

1. Push this repository to GitHub
2. Go to Settings > Pages
3. Select the main branch as the source
4. Your site will be live at `https://yourusername.github.io/pinnacle-berkeley/`

## Local Development

To run locally, use any static file server:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000` in your browser.
